
== install
----
nvm use latest
node -e "fs.writeFileSync('package.json', '{}')"
npm i -D -E @antora/cli@3.1
npm i -D -E @antora/site-generator@3.1
npx antora -v
----
https://docs.antora.org/antora/latest/install/install-antora/
